# 被动

## 创建
- `右键菜单` -> `+被动`

## 参数
| 参数 | 描述 |
| --- | ----------- |
| 名字 | 输入被动名字 |
| 被动描述 | 游戏中显示的被动描述 |
| 种类 | 被动的触发种类，包括[战斗](./combat.html)开始时触发，攻击时触发，受攻击时触发，濒死时触发（当[战斗](./combat.html)为[死斗](./combat.html#死斗)时），行动开始时触发，行动结束时触发 |
| 技能单元队列 | `点击+按钮添加技能单元`，技能效果由多个技能单元组成，发动时由上往下依次发动，拖动可排序。见[技能单元](./spell-unit.html) |
